# Sprite Slicer

一个基于 Web 的交互式图片切片工具，支持实时预览、自动去背和批量下载。

##  功能特点

### 核心功能
- **自定义网格切片**: 支持自定义行数和列数，将大图切割为多个小图。
- **实时预览**: 提供大图网格预览和单个切片效果预览，所见即所得。
- **批量下载**: 支持将所有切片一键打包为 ZIP 文件下载。

### 图像处理
- **自定义颜色去背**: 支持选择特定颜色作为背景进行透明化处理。
- **吸管工具 (Eyedropper)**: 可直接在大图预览区域点击选取目标颜色。
- **容差调节**: 采用欧几里得距离算法，支持 0-200 范围的容差调节，实现精准控色。
- **边缘擦除 (Trim)**: 支持对每个切片的上下左右边缘进行像素级的透明化处理。
- **棋盘格背景**: 预览区域使用经典棋盘格背景，方便观察透明效果。

## 🚀 使用方法

1. **上传图片**
   - 点击 "1. 上传图片" 选择本地图片。

2. **设置网格**
   - 在 "2. 切片网格" 中输入需要的行数和列数。

3. **边缘调整 (可选)**
   - 在 "3. 边缘擦除" 中设置需要透明化的像素值（上、下、左、右）。

4. **背景处理 (可选)**
   - 勾选 "4. 开启去背"。
   - **选取颜色**: 可以直接使用颜色选择器，或点击 "🧪 吸管工具" 后在大图中点击取色。
   - **调整容差**: 拖动容差滑块，通过右侧的 "单个切片效果预览" 窗口实时调整去背强度。

5. **生成与下载**
   - 点击 "开始生成所有切片" 查看所有切割后的图片。
   - 点击 "📥 打包下载 (ZIP)" 获取所有切片文件。

##  技术实现

- **HTML5 Canvas**: 用于图像的切割、像素处理和去背逻辑。
- **JSZip**: 实现浏览器端的切片文件打包功能。
- **Vanilla JavaScript**: 纯原生 JS 实现，无重度框架依赖。
