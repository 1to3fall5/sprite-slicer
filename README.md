# Sprite Slicer Pro

一个基于 Web 的交互式图片切片工具，专为游戏开发者和设计师打造。支持实时预览、自动去背、TGA 格式支持以及批量下载。

## ✨ 功能特点

### 核心功能
- **自定义网格切片**: 支持自定义 **宽 (列) x 高 (行)**，将大图切割为多个小图。
- **智能去背**: 支持选择特定颜色进行透明化处理，配备吸管工具和容差调节。
- **格式支持**: 原生支持 **PNG/JPG** 以及游戏开发常用的 **TGA** 格式。
- **实时预览**: 
  - **无边框悬浮窗**: 点击缩略图可查看大图，支持无边框沉浸式预览。
  - **交互控制**: 支持缩放、拖拽移动、双击复位等手势操作。
- **批量与单个下载**: 
  - 支持一键打包所有切片为 **ZIP** 文件。
  - 单个切片支持独立下载或复制到剪贴板。

### 用户体验
- **现代化 UI**: 采用深色主题 (Dark Mode)，符合专业工具审美。
- **快捷操作**: 支持 Ctrl+V 粘贴图片，拖拽上传。
- **性能优化**: 使用 DocumentFragment 和防抖技术，确保在大量切片下依然流畅。

## 🛠️ 文件结构

项目已从单文件重构为模块化结构，便于维护：

- **index.html**: 核心 HTML 结构。
- **style.css**: 所有样式定义 (UI, 动画, 响应式)。
- **script.js**: 核心逻辑 (Canvas 处理, 事件监听, 文件解析)。
- **refactor.js**: 通用拆分工具，可将 HTML 内联代码自动提取为 CSS/JS 文件。

## 🚀 开发与使用

### 启动项目
推荐使用简单的 HTTP 服务器运行：

```bash
npx http-server .
```

### 使用拆分工具
如果你修改了 HTML 并希望再次分离代码，或者用于其他项目：

```bash
# 默认处理 index.html
node refactor.js

# 处理指定文件
node refactor.js other_file.html
```

## 📦 依赖库
- **JSZip**: 用于生成 ZIP 包。
- **TGAImage**: 用于解析 TGA 格式图片。
